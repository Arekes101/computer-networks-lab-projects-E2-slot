## Project Title
Network Packet Analyzer

## Project Description
We developed a comprehensive network packet analysis tool that helps network administrators monitor traffic, detect anomalies, and analyze network performance in real-time. The tool provides both command-line and web-based interfaces for different user preferences.

## Technology Stack
- **Programming Language**: Python 3.8+
- **Frameworks/Libraries**: Scapy, Flask, Matplotlib, Pandas
- **Tools**: Wireshark (for comparison), Git, Docker

## Key Features Implemented
1. Real-time packet capture from specified network interfaces
2. Protocol analysis and classification
3. Traffic visualization with interactive charts
4. Export functionality for reports
5. Custom filtering rules

## Setup Instructions

### Prerequisites
